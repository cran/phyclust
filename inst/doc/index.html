Announcement of phyclust
Instead of the irregular maintenance for the vignette, phyclust.pdf, we may only update more examples and the newest documents on our website. Please check Phylogenetic Clustering at

http://thirteen-01.stat.iastate.edu/snoweye/phyclust/

Some documents for main functions in the ./Documents/ subdirectory are provided for references and information for outsourced programs ms, seq-gen, and Hap-Clustering. Here are the descriptions for each file.

Seq-Gen.v.1.3.2/ A online directory contains the original document for seq-gen, and it is in HTML format.
Click here to read.
00_Readme-ms.txt A text file for ms indicates the version and documentations.
00_Readme-phyclust-c.txt A text file for the library phylust-c, a kernel of this package, and basically it is in the format of DESCRIPTION file in R and records the version. Only the stable version of C library would be used to build the package.
00_Readme-seq-gen.txt A text file for seq-gen indicates the version and documentations.
msdoc.pdf An online PDF file, the original document for ms.
Click here to read.
ms-Readme.txt A text file, the original readme file in ms.
phyclust-guide.pdf A user guide for phyclust.
seq-gen-Readme.txt A text file, the original readme file in seq-gen.
tzeng-Readme.txt A text file, the original readme file in Hap-Clustering.
